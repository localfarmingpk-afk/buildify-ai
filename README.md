# Buildify AI Assistant (Minimal Starter)

This is a minimal, ready-to-develop Vite + React + TypeScript project exported from the Buildify preview.

## How to run locally

1. Install dependencies:
```
npm install
```

2. Start development server:
```
npm run dev
```

3. Create production build:
```
npm run build
```
Upload the `dist/` folder to Netlify / Vercel / GitHub Pages.
